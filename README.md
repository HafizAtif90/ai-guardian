# AI Guardian - Personal Safety Intelligence Platform

AI-powered personal safety companion with multi-modal threat analysis.

## Features
- Multi-modal threat analysis (Image, Video, Audio, Text)
- AI-powered safety briefings via Google Gemini
- Safe route navigation with live location
- Emergency contacts with universal emergency numbers
- Real-time chat interface for safety queries
- Modern neon/glass UI design

## Tech Stack
- **Frontend**: Next.js 14, TypeScript, Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express
- **AI**: Google Gemini API
